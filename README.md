# Deep Learning Projects

This repository contains **deep learning projects implemented in Python**, focused on solving real-world problems using neural networks and modern deep learning techniques.

The projects demonstrate end-to-end workflows commonly used in **data science, machine learning, and deep learning roles**, including data preprocessing, model architecture design, training, evaluation, and interpretation.

---

## Core Focus

- Image classification and computer vision
- Neural network architecture design
- Model training and evaluation
- Handling real-world datasets and class imbalance
- Performance analysis and model optimization

---

## Projects

Each project is organized in its own folder and includes:
- A dedicated `README.md`
- Notebooks or scripts
- Clear problem statements, methodology, and results
